# Aplikacja blog zbudowana przy użyciu framework'a Django.

Przedstawiony projekt bloga zbudowany został przy użyciu głównych komponentów frameworka Django.
Używając komponentów takich jak: modele, szablony, widoki i adresy URL można stworzyć w pełni funkcjonalną aplikację.
Użytkownicy mogą przeglądać, czytać posty umieszczane przez administratora. Mają też możliwość polecenia innym znajomym 
zamieszczonego posta poprzez wiadomość e-mail. Mogą również za pomocą wyszukiwarki filtrować posty o interesującej ich treści.


## Główne punkty projektu:
1. Utworzenie środowiska wirtualnego w Pythonie.
2. Instalacja Django.
3. Utworzenie i skonfigurowanie projektu Django.
4. Utworzenie aplikacji Django.
5. Tworzenie modeli danych.
6. Tworzenie witryny administracyjnej dla modeli.
7. Tworzenie widoków listy i szczegółów.
8. Tworzenie szablonów.
9. Rozbudowa aplikacji o zaawansowane funkcje.
