# Aplikacja blog zbudowana przy użyciu framework'a Django.

Przedstawiony projekt powstał przy użyciu głównych komponentów framework'a, które pozwalają na stworzenie w pełni funkcjonalnej aplikacji.
Wymienione komponenty to: modele, szablony, widoki i adresy URL. Działanie aplikacji powala użytkownikom na czytanie umieszczonych przez
administratora postów. Daje możliwość dodawania komentarzy, subskrybowania postów na kanale RSS, a także pozwala na filtrowanie i zaawansowane wyszukiwanie
określonych postów. Wszystkie etapy powstawania aplikacji przedstawiłem w poniższym opisie.
